Comment Culture: Reviews vs. Word Counts on Fanfiction.net

This is a subset of data from a late 2018 scrape of the fanfiction archive Fanfiction.net. The full dataset can be found here.
The subset selected is fanfiction written for the video game series Phoenix Wright: Ace Attorney.

Please use the filter selection below to select a rating. This is an indication of the fic's content, and for which ages it is appropriate.

K: Safe for kids
K+: Parental guidance suggested
T: Only appropriate for teens and above
